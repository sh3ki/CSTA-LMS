📚 PROJECT OVERVIEW PROMPT
Colegio De Sta. Teresa De Avila
Learning Management System (LMS)
(Inspired by Google Classroom System)
________________________________________
1️⃣ PROJECT SUMMARY
Develop a Web-Based Learning Management System (LMS) for Colegio De Sta. Teresa De Avila, inspired by the clean, modern, and minimal design of Google Classroom.
The system will allow:
•	Administrators to manage teachers, students, classes, and subjects
•	Teachers to manage subjects, resources, tasks, and announcements
•	Students to access tasks and announcements
The system must follow a role-based access control structure with three roles:
•	Admin
•	Teacher
•	Student
________________________________________
2️⃣ TECHNOLOGY STACK
•	Backend Framework: PHP Laravel (Latest Version)
•	Frontend: HTML5, CSS3, JavaScript
•	Database: MySQL
•	Authentication: Laravel Authentication System
•	UI Structure: Responsive Admin Dashboard Layout
